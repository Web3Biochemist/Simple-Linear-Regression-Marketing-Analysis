# Simple Linear Regression – Marketing ROI Analysis

## Project Overview

This project applies Simple Linear Regression using Python and Statsmodels to determine which marketing channel has the greatest impact on sales. The analysis includes data cleaning, exploratory data analysis (EDA), regression modeling, model evaluation, diagnostic testing, and business recommendations.

## Objectives

- Load and clean the marketing dataset
- Perform exploratory data analysis
- Identify the marketing variable most strongly correlated with Sales
- Build an Ordinary Least Squares (OLS) regression model
- Evaluate model performance
- Validate regression assumptions
- Recommend the marketing channel with the highest ROI

## Dataset

The dataset contains the following variables:

- TV
- Radio
- Social_Media
- Sales

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels
- Scikit-learn

## Results

The analysis identified TV advertising as the strongest predictor of sales.

Performance metrics:

- R² = 0.9990
- RMSE = 2.88
- MAE = 2.31

The regression model demonstrated excellent predictive performance and satisfied the major assumptions of linear regression.
